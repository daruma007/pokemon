# pokemon-dataset

スクレイピングの練習として、剣盾までのポケモン図鑑を作成しました。
pokemon-dataset.csv

現時点では、以下のカラムが含まれています。
#### ・#
図鑑No
#### ・Name
英語名
#### ・Type
タイプ
#### ・Total
合計種族値
#### ・HP
HPの種族値
#### ・Attack	
攻撃の種族値
#### ・Defense
防御の種族値
#### ・Sp. Atk
特攻の種族値
#### ・Sp. Def
特防の種族値
#### ・Speed
素早さの種族値  

今後は、特性、日本語名を追加する予定です。

参照URL
https://pokemondb.net/pokedex/all
